# Dubai-Landmarks-Blog

## Description

> A simple blog demonstrating various landmarks of Dubai. Built with NodeJS, Parse-Server and Angular 13.

----------------------------------------------------------------------------------------------------

## Built With

* [Node.js](https://nodejs.org/en/) - v12.22.5
* [Parse-Server](https://github.com/parse-community/parse-server) - v4.10.4
* [Parse-Dashboard](https://github.com/parse-community/parse-dashboard) - v3.3.0
* [Express.js](https://expressjs.com/) - v4.17.1
* [MongoDB - Atlas](https://www.mongodb.com/atlas/database) - v4.4.10
* [Angular](https://angular.io/) - v13.1.0
* [Bootstrap](https://getbootstrap.com) - v4.6.1

----------------------------------------------------------------------------------------------------

## Prerequisites

- NodeJS, you can install it from [here](https://nodejs.org).
- Angular CLI, you can install it from [here](https://angular.io/guide/setup-local).

----------------------------------------------------------------------------------------------------

## Installation

- Clone the project: `git clone git@github.com:Gkonst1/Dubai-Landmarks-Blog.git`
- Navigate at the projects root directory: `cd Dubai-Landmarks-Blog`

### For the Backend:
> Open a terminal in project's `/backend` directory.
- Run the command `npm install`.
- Add the `.env` file in the `/backend` directory.

### For the Frontend:
> Open a terminal in project's `/frontend` directory.
- Run the command `npm install`.

*In case you get EACCES permissions error, from the `npm install` command, try running it as an administrator.*

----------------------------------------------------------------------------------------------------

## Usage

### For the Backend:
- Start the server with the command `npm start`.
- After that you can visit the server dashboard at `http://localhost:7777/dashboard`.

### For the Frontend:
- Start the application with the command `ng serve`.
- After that you can find the blog at `http://localhost:4200`.
